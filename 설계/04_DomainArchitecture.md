# Purpose

이 문서는 ASTER의 시스템을 기능별 도메인으로 분리하여 책임과 의존성을
정의한다. 모든 개발은 도메인 단위로 진행한다.

# Domain Definitions

## Authentication

책임 - 회원가입 - 로그인 - 인증 - 권한 관리

의존 - User

------------------------------------------------------------------------

## Project

책임 - 프로젝트 생성 - 저장 - 버전 관리 - 즐겨찾기

의존 - Brand Brief - Generation

------------------------------------------------------------------------

## Brand Interview

책임 - AI 질문 생성 - 사용자 답변 수집 - 브랜드 정보 구조화

출력 - Interview Result

------------------------------------------------------------------------

## Aster Brain

책임 - 인터뷰 분석 - 업종 분석 - 브랜드 핵심 정보 추출 - Brand Brief
생성

입력 - Interview Result

출력 - Brand Brief

------------------------------------------------------------------------

## Brand Brief

책임 - 브랜드명 - 업종 - 핵심 가치 - 타깃 - 키워드 - 브랜드 목표

------------------------------------------------------------------------

## Brand Strategy

책임 - 포지셔닝 - 톤앤매너 - 브랜드 성격 - 추천 방향

------------------------------------------------------------------------

## Style

책임 - 스타일 라이브러리 - 스타일 추천 - 카테고리 관리

------------------------------------------------------------------------

## Prompt

책임 - 내부 프롬프트 자동 생성 - 모델별 프롬프트 변환

------------------------------------------------------------------------

## Generation

책임 - AI 모델 호출 - 컨셉 이미지 생성 - 생성 이력 관리

------------------------------------------------------------------------

## One Click Edit

책임 - 심플화 - 컬러 변경 - 타이포 변경 - 스타일 변경

------------------------------------------------------------------------

## Concept Board

책임 - 컬러 팔레트 - 키워드 - 브랜드 설명 - 무드보드 구성

------------------------------------------------------------------------

## Mockup

책임 - 명함 - 간판 - 패키지 - 디지털 목업

------------------------------------------------------------------------

## Design Memory

책임 - 사용자 선호 저장 - 개인화 추천

------------------------------------------------------------------------

## Subscription

책임 - 플랜 관리 - 생성량 제한 - 결제 상태

------------------------------------------------------------------------

## User

책임 - 프로필 - 설정 - 프로젝트 목록

------------------------------------------------------------------------

## Admin

책임 - 사용자 관리 - 스타일 관리 - 통계 - 신고 처리

------------------------------------------------------------------------

# Domain Rules

-   각 도메인은 단일 책임(SRP)을 가진다.
-   도메인 간 직접 결합을 최소화한다.
-   Aster Brain을 중심으로 Brand Brief를 생성한다.
-   Prompt 도메인만 AI 모델과 직접 연결된다.
