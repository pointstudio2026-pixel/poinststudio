# Product Overview

Aster는 디자이너를 위한 AI 브랜딩 플랫폼이다.

핵심 원칙은 '브랜드를 이해한 후 방향성을 설계하고, 그 다음 참고용 컨셉을
생성한다.'이다.

------------------------------------------------------------------------

# Core Workflow

Project → AI Brand Interview → Aster Brain → Brand Brief → Brand
Strategy → Style Engine → Prompt Engine → AI Model Router → Image
Generation → One Click Editor → Concept Board → Mockup → Export →
Project Save

------------------------------------------------------------------------

# Core Engines

## Project Engine

프로젝트 생성, 저장, 버전 관리.

## AI Brand Interview

브랜드 정보를 질문으로 구체화한다.

## Aster Brain

입력 정보를 분석하여 구조화된 Brand Brief를 생성한다.

## Brand Strategy Engine

브랜드 가치, 성격, 톤앤매너, 포지셔닝을 도출한다.

## Style Engine

150개 스타일 라이브러리에서 적합한 스타일을 추천한다.

## Prompt Engine

사용자 입력 없이 내부 프롬프트를 자동 생성한다.

## AI Model Router

여러 이미지 생성 API를 교체 가능한 구조로 연결한다.

## Image Generation

참고용 브랜드 컨셉 이미지를 생성한다.

## One Click Editor

심플화, 컬러 변경, 타이포 변경 등 빠른 수정 기능.

## Concept Board

컬러, 타이포, 스타일 키워드, 설명을 하나의 보드로 구성.

## Mockup Engine

명함, 간판, 패키지 등 참고용 목업 생성.

## Design Memory

사용자 자신의 사용 이력을 기반으로 선호 스타일을 개인화 추천한다.

# Data Flow

1.  프로젝트 생성
2.  브랜드 인터뷰
3.  Brand Brief 생성
4.  Brand Strategy 생성
5.  스타일 선택
6.  이미지 생성
7.  원클릭 수정
8.  컨셉보드 생성
9.  목업 생성
10. 프로젝트 저장

------------------------------------------------------------------------

# Product Rules

-   AI는 디자이너를 대체하지 않는다.
-   생성 결과는 참고용 컨셉이다.
-   최종 디자인은 사람이 완성한다.
-   저작권과 상표권을 존중한다.
-   AI 모델은 교체 가능한 구조(Adapter)로 설계한다.
