PRD-015

Design Memory

Priority

P1 (Personalization Engine)

------------------------------------------------------------------------

# Goal

Design Memory는 사용자의 작업 이력을 기반으로 개인화된 추천을 제공한다.

사용자의 데이터만 활용하여 선호 스타일과 작업 패턴을 학습하며, 최종
결정은 항상 사용자가 내린다.

------------------------------------------------------------------------

# Principles

-   사용자 데이터를 기본으로 사용한다.
-   자동 적용하지 않고 추천만 제공한다.
-   언제든 초기화하거나 비활성화할 수 있다.

------------------------------------------------------------------------

# Inputs

-   Brand Brief
-   Brand Strategy
-   선택한 스타일
-   One Click Edit 이력
-   즐겨찾기
-   프로젝트 메타데이터

------------------------------------------------------------------------

# Stored Preferences

-   선호 스타일
-   선호 컬러 방향
-   선호 타이포 방향
-   자주 사용하는 목업
-   자주 사용하는 수정 액션

------------------------------------------------------------------------

# JSON Schema

``` json
{
  "favoriteStyles": [],
  "favoriteColors": [],
  "favoriteTypography": [],
  "favoriteMockups": [],
  "favoriteEditActions": []
}
```

------------------------------------------------------------------------

# Recommendation Rules

-   최근 작업 가중치 우선
-   반복 선택은 신뢰도 증가
-   현재 Brand Strategy와 충돌하면 추천만 제공

------------------------------------------------------------------------

# Privacy

-   프로젝트별 데이터 분리
-   사용자 요청 시 즉시 삭제
-   개인 데이터는 추천 외 목적으로 사용하지 않는다.

# Claude Code Instructions

Design Memory는 추천 엔진이며 자동 변경 엔진이 아니다. 모든 추천은
사용자의 명시적 선택 이후에만 적용한다.
