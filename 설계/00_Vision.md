# Mission

> **Design Begins with Direction.**

Aster는 이미지를 생성하는 서비스가 아니라, 디자이너가 더 빠르게 브랜드의
방향성을 탐색하도록 돕는 **AI 브랜딩 플랫폼**이다.

------------------------------------------------------------------------

# Vision

세계 최고의 **AI Branding Platform**을 구축한다.

Aster는 브랜드 브리프부터 전략, 컨셉 탐색, 참고용 이미지, 컨셉보드,
목업까지 하나의 워크플로로 연결하는 플랫폼을 목표로 한다.

------------------------------------------------------------------------

# Core Principles

1.  AI는 디자이너를 대체하지 않는다.
2.  최종 디자인은 항상 사람이 완성한다.
3.  우리는 이미지를 생성하는 것이 아니라 브랜드의 방향성을 설계한다.
4.  프롬프트보다 브랜드 사고를 먼저 설계한다.
5.  저작권·상표권을 존중하는 구조를 기본으로 설계한다.
6.  모든 생성 결과는 참고용 컨셉이며 수정 가능해야 한다.
7.  입력은 최소화하고 선택 중심 UX를 제공한다.

------------------------------------------------------------------------

# Product Identity

Aster는 AI 이미지 생성기가 아니다.

Aster는 디자이너를 위한 AI Branding Assistant이며, 브랜드를 이해하고
방향성을 제안하는 워크플로 플랫폼이다.

------------------------------------------------------------------------

# Product Goals

-   브랜드 방향성을 3분 이내 제안
-   첫 컨셉 탐색을 15분 이내 완료
-   디자이너의 아이디어 탐색 시간을 획기적으로 단축

------------------------------------------------------------------------

# Non Goals

Aster는 다음을 목표로 하지 않는다.

-   디자이너를 대체하는 서비스
-   특정 브랜드를 모방하는 서비스
-   저작권·상표권 침해를 유도하는 서비스
-   프롬프트 작성 자체를 핵심 가치로 삼는 서비스

------------------------------------------------------------------------

# Success Metrics

-   첫 컨셉 도출 시간
-   프로젝트 완료 시간
-   무료 → 유료 전환율
-   구독 유지율
-   사용자당 절약한 작업 시간

------------------------------------------------------------------------

# Brand Promise

Aster는 AI로 디자인를 대신하지 않는다.

Aster는 디자이너가 더 좋은 방향을 더 빠르게 찾을 수 있도록 돕는다.
