# Purpose

본 문서는 ASTER의 전체 PRD를 관리하는 인덱스이다.

PRD는 하나의 거대한 문서가 아니라 기능 단위 문서로 분리한다.

# Standard Template

각 PRD 문서는 반드시 아래 항목을 포함한다.

-   목적
-   사용자
-   화면
-   입력
-   출력
-   비즈니스 규칙
-   예외 처리
-   API
-   Database
-   AI Agent
-   Acceptance Criteria
-   Definition of Done

# Development Rule

Claude Code는 반드시 해당 PRD 문서를 먼저 읽은 후 구현을 시작한다. 구현
중 문서와 충돌하는 사항은 임의로 결정하지 않는다.
